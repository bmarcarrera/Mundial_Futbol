# ⚽ Mundial_Futbol

[![Open In Colab](https://google.com/bmarcarrera/Mundial_Futbol/blob/main/Ganadores_de_mundiales_y_caracteristicas.ipynb)]

¡Hola! Te doy la bienvenida a mi repositorio. Aquí encontrarás contenido relacionado con todos los Mundiales de Fútbol desde su creación hasta 2022. 

Por el momento, he incluido un archivo con los ganadores de cada edición junto a diferentes características asociadas. Iré subiendo más contenido y análisis a medida que lo vaya elaborando.

## 🚀 Cómo puedes ejecutar este proyecto

Puedes visualizar y ejecutar todo mi análisis de forma interactiva con un solo clic en tu propio navegador:

1. Haz clic en el botón **Open in Colab** que ves aquí arriba.
2. Una vez dentro de la barra superior de Colab, selecciona **Entorno de ejecución** ➔ **Ejecutar todas**.

## 📁 Estructura del Repositorio

* `Mundial.csv`: Archivo con el registro histórico de los campeones del mundo y variables asociadas (configurado para cargarse de forma remota en el código mediante URL Raw).
* `Ganadores_de_mundiales_y_características.ipynb`: Mi cuaderno de Google Colab con el procesamiento de datos, estadísticas y gráficos interactivos.

## 🛠️ Tecnologías y Librerías que utilicé

* **Python 3**
* **Pandas** (Para la estructuración y manipulación de los datos históricos)
